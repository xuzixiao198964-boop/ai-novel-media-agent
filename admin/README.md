# 管理后台系统

AI小说视频生成系统的管理后台，提供完整的系统管理功能。

## 功能模块

1. **数据概览** - 实时统计数据和趋势图表
2. **用户管理** - 用户列表、状态管理、套餐管理
3. **小说管理** - 小说作品审核、上下架管理
4. **视频管理** - 视频作品查看和管理
5. **任务监控** - 实时任务状态监控和控制
6. **API Key管理** - API密钥创建、吊销和监控
7. **财务报表** - 收入成本分析和趋势图表
8. **发布管理** - 内容发布记录和重试
9. **系统日志** - 系统运行日志查询
10. **系统配置** - 套餐定价和系统参数配置

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand (状态管理)
- React Router (路由)
- Recharts (图表)
- Axios (HTTP客户端)

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

构建后的文件在 `dist/` 目录，可以部署到任何静态文件服务器。

## 默认登录

- 用户名: admin
- 密码: (需要在后端配置)

## API配置

后端API地址在 `vite.config.ts` 中配置，默认代理到 `http://localhost:8000`
