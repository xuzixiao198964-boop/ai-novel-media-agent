## 🔧 Systemd 服务配置

### media-agent.service (前端服务)

```ini
[Unit]
Description=AI Media Agent Frontend Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/media-agent/frontend
ExecStart=/usr/bin/npm run start
Restart=always
RestartSec=3
Environment="NODE_ENV=production"
Environment="PORT=8000"

[Install]
WantedBy=multi-user.target
```

### ai-novel-agent.service (后端服务)

```ini
[Unit]
Description=AI Novel Agent Backend Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/ai-novel-agent/backend
ExecStart=/usr/bin/python3 -m uvicorn app.main:app --host 0.0.0.0 --port 9000
Restart=always
RestartSec=3
Environment="PYTHONPATH=/opt/ai-novel-agent/backend"
Environment="LLM_API_KEY=sk-9fcc8f6d0ce94fdbbe66b152b7d3e485"
Environment="LLM_API_BASE=https://api.deepseek.com/v1"
Environment="LLM_MODEL=deepseek-chat"

[Install]
WantedBy=multi-user.target
```

## 🚀 部署流程

### 1. 准备环境变量

```bash
# Windows PowerShell
$env:DEPLOY_SSH_PASSWORD = "your-ssh-password"
$env:LLM_API_KEY = "sk-your-deepseek-key"

# Linux/Mac
export DEPLOY_SSH_PASSWORD='your-ssh-password'
export LLM_API_KEY='sk-your-deepseek-key'
```

### 2. 执行部署脚本

```bash
# 从本地部署到服务器
cd E:\work\ai-novel-agent

# 使用Systemd部署（推荐）
python deploy/deploy_systemd_ssh.py

# 或使用Docker部署
python deploy/deploy_ssh.py
```

### 3. 手动部署步骤

#### 后端部署

```bash
# 1. SSH连接到服务器
ssh root@104.244.90.202

# 2. 创建部署目录
mkdir -p /opt/ai-novel-agent/backend
cd /opt/ai-novel-agent

# 3. 上传代码（从本地）
# 使用scp或git clone
scp -r backend/* root@104.244.90.202:/opt/ai-novel-agent/backend/

# 4. 安装依赖
cd /opt/ai-novel-agent/backend
pip3 install -r requirements.txt

# 5. 配置环境变量
cat > .env << EOF
LLM_API_KEY=sk-9fcc8f6d0ce94fdbbe66b152b7d3e485
LLM_API_BASE=https://api.deepseek.com/v1
LLM_MODEL=deepseek-chat
HOST=0.0.0.0
PORT=9000
EOF

# 6. 创建systemd服务
sudo cp systemd/ai-novel-agent.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable ai-novel-agent
sudo systemctl start ai-novel-agent

# 7. 检查服务状态
sudo systemctl status ai-novel-agent
```

#### 前端部署

```bash
# 1. 本地构建
cd E:\work\ai-novel-agent\mobile-app
npm install
npm run build

# 2. 上传到服务器
scp -r dist/* root@104.244.90.202:/opt/media-agent/frontend/dist/

# 3. 重启前端服务
ssh root@104.244.90.202 "sudo systemctl restart media-agent"
```

### 4. 验证部署

```bash
# 检查服务状态
ssh root@104.244.90.202 "systemctl status ai-novel-agent"
ssh root@104.244.90.202 "systemctl status media-agent"

# 测试API
curl http://104.244.90.202:9000/api/health

# 测试前端
curl http://104.244.90.202:8000/
```
