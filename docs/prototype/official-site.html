🔗 OpenClaw 接入指南

本平台实现 OpenClaw 1.0 协议，允许第三方系统：

  • ✓ 查询平台能力（/openclaw/capabilities）
  • ✓ 创建生成任务（/openclaw/tasks）
  • ✓ 轮询任务进度（/openclaw/tasks/{id}）
  • ✓ 获取产出文件（/openclaw/tasks/{id}/output）

帮助中心

如何创建第一个小说生成任务？

注册并登录后，系统会引导您选择创作方式。选择"只生成小说"后，设置篇幅和题材（也可选随机），确认后系统即刻开始 AI 创作。

如何将视频自动发布到抖音？

在"设置 → 平台绑定"中绑定您的抖音账号（通过 OAuth 授权）。之后在视频作品页点击"发布抖音"即可。专业版及以上套餐支持自动发布。

收费是如何计算的？

采用充值余额模式。小说按千字计费，视频按秒计费，具体单价取决于您选择的套餐。每笔消费会在"消费记录"中详细展示。

OpenClaw 如何接入？

首先在官网申请 API Key，然后参照开发者中心的 OpenClaw 接入指南，使用标准 HTTP 请求即可对接。详见 API 文档。
