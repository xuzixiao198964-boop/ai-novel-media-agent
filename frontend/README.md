# Frontend - 用户端 Web 应用

React + TypeScript + Vite 实现的用户端应用

## 启动方式

```bash
# 安装依赖
npm install

# 开发环境 (http://localhost:3000)
npm run dev

# 构建生产版本
npm run build
```

## 功能页面

- `/login` - 登录页面
- `/dashboard` - 仪表盘
- `/create` - 创建任务
- `/tasks` - 任务列表
- `/novels` - 作品列表

## API 连接

所有 API 请求会自动代理到 `http://localhost:8000`

## 技术栈

- React 18
- TypeScript
- React Router
- Vite
